# WebCLI SDK
[![npm version](https://img.shields.io/npm/v/@kaoruisaac/webcli.svg)](https://www.npmjs.com/package/@kaoruisaac/webcli)
Browser SDK for connecting a web app to the WebCLI Chrome Extension and Desktop Runtime.

```ts
import { Webcli } from "@kaoruisaac/webcli";
```

See the repository README for full setup and usage documentation.
